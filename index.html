Javelin Bootstrap

A set of UI elements styled using Twitter Boostrap toolkit to use with Javelin

Common Includes

All plugins require the following minimum includes on a page to work.

<script src="javelin/pkg/init.min.js" type="text/javascript"></script>
<script src="javelin/pkg/javelin.dev.js" type="text/javascript"></script>
<script>
  var __DEV__ = 1;
</script>
