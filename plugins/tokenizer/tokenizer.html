Tokenizer

This plugin is directly borrowed of Javelin source itself. It is illustrated here for documentation and cataloging purposes. Take a look at javelin/pkg/typeahead.dev.js

A tokenizer is a UI component similar to a text input, except that it allows the user to input a list of items ("tokens"), generally from a fixed set of results. A familiar example of this UI is the "To:" field of most email clients, where the control autocompletes addresses from the user's address book.

Required Includes

<link rel="stylesheet" href="plugins/tokenizer/tokenizer.css" type="text/css" />
<script src="plugins/typeahead/typeahead.dev.js" type="text/javascript"></script>
<script src="plugins/tokenizer/tokenizer.js" type="text/javascript"></script>

Required Markup

<div id="some selector" style="position: relative;">
  <input type="text" data-sigil="tokenizer-input" />
</div>
